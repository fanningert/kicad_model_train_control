# Model Train Control

Control unit for analog model trains, based on ESPHome.

Supported szenarios:

- Power line
- Train Switch
- Signal
- Light
- Uncoupler

## Parts

* pcb_hbridge - HBridge with is used as a three state switch or as a variable power control for the power line.
* pcb_main - Main pcb for the project
